# 更新日志

## [1.0.3] - 2026-06-08

### 新增
- 支持 JPEG 压缩质量配置。
- 新增 `getResults()`，支持批量压缩逐项返回成功或失败结果。
- `CompressResult` 新增 `sourcePath`，用于定位原始输入路径或 URI。
- 支持批量压缩最大并发数配置。

### 优化
- 对齐 Luban Android 当前压缩参数计算策略。
- 优化 URI 预处理和临时文件清理流程。
- 优化 EXIF 方向处理和图片解码流程。
- 压缩后文件不小于原图时回退复制原图，避免输出变大。
- 更新 README，补充初始化、质量调节、批量失败语义和缓存清理说明。

### 新增
- 🎉 首次发布 Luban 图片压缩库的鸿蒙版本

- 📊 提供完整的压缩过程回调机制
- 🎨 支持透明通道保留配置
- 📁 支持自定义输出目录
- 🔍 支持文件重命名回调
